確率統計

期待値（離散型確率変数）
\[ E[X] = \sum_{i=1}^\infty x_{i}P(X=x_{i}) \]
\[ E[aX + bY] = aE[X] + bE[Y] \]

分散
\[ s^2 = \frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2 \]
  • ここで $\bar{x}$ は平均値を表す。
\[ V[X] = E[X^2] - (E[X])^2 \]
\[ V[X + b] = V(X) \]
\[ V[aX] = a^2 V(X) \]

標準偏差
\[ s = \sqrt{\frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2} \]
  • ここで $\bar{x}$ は平均値を表す。

積率母関数
  • 確率変数 $X$ の積率母関数またはモーメント母関数は、期待値が存在するならば次の式で定義される。
\[ M_X(t) := E(e^{tX}), \quad t \in\Bbb R \]
\[ E(X^n) = M^{(n)}_X(0) = \frac{d^nM_X}{dt^n}(0) \]

二項分布
  • 結果が成功か失敗のいずれかである試行を独立に $n$ 回行ったときの成功回数を確率変数とする離散確率分布。ただし、各試行における成功確率 $p$ は一定とする。
\[ E[X] = np \]
\[ V[X] = np(1 - p) \]

指数分布
  • 指数分布は台 $(0, \infty)$ を持ち、母数 $\lambda \gt 0$ に対して確率密度関数が
\[ f(x; \lambda) = \lambda e^{- \lambda x} \]
  • で与えられる。
\[ E(x) = \frac{1}{\lambda} \]
\[ V(x) = \frac{1}{\lambda^2} \]

共分散
  • 大きさが同じ2つのデータの間での、平均からの偏差の積の平均値。
\[ Cov[X, Y] = E[(X - E[X])(Y - E[Y])] \]
\[ Cov[X, Y] = E[(XY)] - E[X]E[Y] \]
  • $X$ と $Y$ の共分散は $\sigma_{XY}$ や $\sigma(X, Y)$ と表記されることもある。
\[ V\left[\sum_i a_i X_i\right] = \sum_{i, j} a_i a_j Cov[X_i, X_j] \]
  • （参考：分散）
